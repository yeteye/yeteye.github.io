欢迎来到this place,我们的任务是学习git 和markdown的使用

What is markdown

Markdown 是一种轻量级的标记语言，它允许你使用简单的文本语法编写结构化的文档。与复杂的 HTML 标签相比，Markdown 更容易学习和使用。Markdown 语法可用于编写各种文档，如 README、博客文章、技术文档、演示文稿等。

学习路线

  1. 首先，你应该学习markdown的语法和格式。这里是一份markdown教程
  2. 接下来是一份git教程,你可以了解一些git指令[^q]。
  3. 了解如何将 Markdown 文件推送到 Git 仓库。你需要了解如何创建仓库、添加文件、提交更改、拉取和合并分支等。
  4. 掌握 Git 的分支和合并功能。这是 Git 最重要的功能之一，它可以帮助你更好地管理代码库。
  5. 最后，了解如何在 GitHub 上使用 Markdown 和 Git。GitHub 是一个非常流行的代码托管平台，它提供了很多功能，如代码托管、问题跟踪、代码审查等。掌握 GitHub 的使用可以帮助你更好地管理你的项目。

必看文档和指南

  1. Markdown 官方文档 - 详细介绍 Markdown 的语法和格式。
  2. Git 官方文档 - Git 的官方文档，详细介绍 Git 的各种功能和命令。
  3. GitHub 官方文档 - GitHub 的官方文档，详细介绍 GitHub 的使用和功能。
  4. GitHub Guides - GitHub 提供的教程和指南，可以帮助你更好地了解 GitHub 的使用和功能。
  5. Markdown 教程 | 菜鸟教程 (runoob.com) - 菜鸟教程的Markdown教程，很棒的中文网站。
  6. Git 教程 | 菜鸟教程 (runoob.com) - 菜鸟教程的git教程，很棒的中文网站。

一些有用的文档

  1. 提问的智慧
  2. Angular提交信息规范 - Git Guide (zj-git-guide.readthedocs.io)
  3. Google 开源项目风格指南——中文版 — Google 开源项目风格指南 (zh-google-styleguide.readthedocs.io)
